# Personal Portfolio Website 🚀

A full-stack personal portfolio website to showcase my skills, projects, and experience.

Built using **React.js**, **Node.js**, **Express.js**, and **MySQL**.

## ✨ Features

- Responsive portfolio website
- Home, About, Projects, and Contact pages
- Projects fetched dynamically from MySQL database
- Add new projects using a form
- REST API integration
- React Router navigation

## 🛠️ Tech Stack

**Frontend**
- React.js
- JavaScript
- HTML
- CSS
- Axios

**Backend**
- Node.js
- Express.js

**Database**
- MySQL
